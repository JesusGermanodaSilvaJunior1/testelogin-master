?? Desafio Prático – Sistema de Login
Objetivo: Criar um sistema simples de login usando PHP, HTML5, CSS3 e estilizaçăo com Bootstrap ou Bulma.
Requisitos:
Criar uma página login.php com:
Formulário contendo campos de usuário e senha.
Botăo de login estilizado com Bootstrap ou Bulma.
Criar um script PHP que:
Valide os dados enviados pelo formulário.
Compare com credenciais armazenadas em um banco de dados MySQL (ex.: tabela usuarios com colunas id, username, password).
Use session_start() para iniciar a sessăo do usuário autenticado.
Redirecione para uma página restrita (dashboard.php) se o login for bem-sucedido.
Exiba mensagem de erro se as credenciais forem inválidas.
Criar a página dashboard.php que:
Mostre uma mensagem de boas-vindas com o nome do usuário logado.
Tenha um botăo de logout que destrua a sessăo e volte para login.php.
